!!!! IMPORTANT !!!! 

ALL RECIPES MUST BE INSIDE THE RECIPES FOLDER AND HAVE THERE OWN .JSON


{
  "_id": "CHR_Recipe_01", --- Can be whatever you choose, Mongold ID is handled 
  "areaType": 6, --- Where the craft will be done, reference supplied list below
  "requiredLevel": 1,--- Required station level
  "productionTime": 60,
  "endProduct": "5c94bbff86f7747ee735c08f", --- Item you will recieve
  "count": 1, --- How many you will get
  "needFuelForAllProductionTime": true, 
  "inputs": [ 
    { "tpl": "5755356824597772cb798962", "count": 1 }, --- Items needed for craft, follow format to add additional items. Get item ID's from here https://db.sp-tarkov.com/search
    { "tpl": "544fb45d4bdc2dee738b4568", "count": 1 }
  ],
  "tools": [
    { "tpl": "590c2d8786f774245b1f03f3", "count": 1 } --- "Tools" required for crafting, can be any item as far as I can tell. 
  ]
}



Use some of these at your own...risk? I haven't tested ones that do not contain vanilla recipes, maybe it'll work, maybe it won't.
